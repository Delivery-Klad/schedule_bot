# [RTU MIREA schedule bot](https://t.me/schedule_memrea_bot)

### Requirements:
| Library          | Version |
|------------------|---------|
| PyTelegramBotAPI | 3.7.6   |
| psycopg2-binary  | 2.8.6   |
| requests         | 2.25.1  |

***
### Heroku variables:
| Variable      | Value                   |
|---------------|-------------------------|
| TOKEN         | Telegram bot token      |
| DB            | Database name           |
| DB_user       | Database user           |
| DB_pass       | Database user password  |
| DB_host       | Database host           |
